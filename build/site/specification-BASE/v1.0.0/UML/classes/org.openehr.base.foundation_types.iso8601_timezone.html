Iso8601_timezone Class

Class

Iso8601_timezone

Description

ISO8601 timezone string, in format:

  • Z | ±hh[mm]

where:

  • hh is "00" - "23" (0-filled to two digits)

  • mm is "00" - "59" (0-filled to two digits)

  • Z is a literal meaning UTC (modern replacement for GMT), i.e. timezone +0000
